## 2. Categories redesign
- Сохранены table semantics на desktop.
- Колонки: Рубрика, Slug, Новостей, Статус, Порядок, Действия.
- Slug оформлен compact monospace badge; news count — numeric badge.
- Active/inactive оформлены спокойными status badges с dot-indicator; inactive не красный.
- Порядок — единая группа `[icon] number [icon]`, bounded disabled states, title и aria-label.
- Edit/open/delete actions переведены на Tabler icon controls; destructive delete остаётся отдельным action.
- Mobile table превращается в compact cards без horizontal overflow.

## 3. Categories responsive
`admin-news-categories-reorder.spec.ts` PASS desktop and mobile. Проверены bounded controls и отсутствие editable `order` input.

## 4. Placements popup redesign
- `+ ещё N` теперь открывает portal popup с заголовком «Все размещения», count «N разделов» и close button.
- Raw placement keys и `All placements` не показываются.
- Каждое placement — отдельная строка с readable Russian label, spacing, hover и переносом длинного текста.
- Для списков >12 добавлен локальный поиск и empty state «Ничего не найдено».
- Popup width `min(420px, calc(100vw - 24px))`, max-height 520px, overflow-x hidden, vertical touch scroll.

## 5. Placements search/65+ scroll
Acceptance runtime seeded a document with 70 placements. Popup implementation renders one row per placement, body-level portal, bounded width and vertical scroll. Existing placement selector scroll spec passed desktop/mobile; portal Escape and focus return remained passing in document regression specs.
